# Веб-сайт кафе «Байкал Буузы»

Проект представляет собой полноценную экосистему для кафе бурятской кухни «Байкал Буузы», включающую в себя ознакомление с меню, авторизацию пользователей, онлайн-заказ доставки и бронирование столиков.

## Стек технологий

- **Бэкенд**: Go (Golang) + `chi` (роутер) + `pgx` (драйвер PostgreSQL) + `bcrypt` (хэширование паролей) + `golang-jwt` (JWT-сессии в HTTP-only cookie).
- **Фронтенд**: Svelte 5 + Vite + TailwindCSS.
- **База данных**: PostgreSQL 15 (запускается в Docker).

## Архитектура бэкенда

Бэкенд спроектирован по принципам **Чистой Архитектуры (Clean Architecture)**:
- **Handlers** (Контроллеры/транспорт): Прием HTTP-запросов, валидация входящих данных.
- **Services** (Бизнес-логика): Основные правила системы.
- **Repositories** (Доступ к данным): SQL-запросы к базе данных PostgreSQL.
- **DTO и Мапперы**: Изолируют внутренние структуры БД от внешнего API.

Все логи, системные сообщения и отладочная информация выводятся в нижнем регистре на русском языке. В коде бэкенда полностью отсутствуют комментарии.

## Запуск проекта

### 1. Подготовка окружения и База данных

Для работы проекта требуется запущенный Docker.

Запустите контейнер с PostgreSQL:
```bash
docker compose up -d
```
Это запустит контейнер `baikal_buuzy_db` на стандартном порту `5432` и создаст базу данных `baikal_buuzy`.

### 2. Запуск бэкенда (Go)

Перейдите в директорию бэкенда:
```bash
cd backend
```

Убедитесь, что файл `.env` содержит корректные параметры подключения:
```env
PORT=8080
DATABASE_URL=postgres://postgres:secretpassword@localhost:5432/baikal_buuzy?sslmode=disable
JWT_SECRET=supersecretjwtkeyforbaikalbuuzycafe2026
```

Запустите бэкенд-сервер:
```bash
go run cmd/app/main.go
```
При первом запуске сервер автоматически применит схему данных из `backend/migrations/schema.sql` и наполнит базу данных начальным меню и категориями.

### 3. Запуск фронтенда (Svelte)

Перейдите в директорию фронтенда:
```bash
cd frontend
```

Установите зависимости (если еще не установлены):
```bash
npm install
```

Запустите сервер для разработки:
```bash
npm run dev
```
Фронтенд будет запущен локально (обычно на порту `5173`).
